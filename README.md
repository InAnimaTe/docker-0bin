docker-0bin
===========

A base debian wheezy with [0bin](https://github.com/sametmax/0bin) installed.

Default settings
================
The default settings.py contains the folowing config :

```
HOST = "0.0.0.0"
PORT = "8000"
PASTE_FILES_ROOT = "/var/zerobin/content"
```
